Audio Story

The goal of this project was to create an audio file that tells a story without words. I enjoyed working on this project because of deep it made my imagination run. This was edited using various sounds I found on the internet and merging them together usign the Audacity application. This was one of my favourite assignments this semester. However, I could have done a lot better in compailing the sounds and noise reduction. This project thaught me a lot about the Audacity application and how different representations of life could be told with just sounds. Listen and create your own story out of this audio.
